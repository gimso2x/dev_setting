# Git 자동 세션 관리 워크플로우

기존 `settings.json`에 정의되어 있던 백그라운드 구동 스크립트를 Antigravity 환경에서 수동/반자동으로 사용할 수 있도록 만든 워크플로우입니다.
세션 작업 시작 시와 종료 시에 다음 셸 스크립트를 실행하여 컨텍스트 일관성을 유지하고 작업 내용을 자동으로 스태이징 및 커밋할 수 있습니다.

## 스크립트의 위치
모든 스크립트는 `.agent/scripts` 에 위치합니다. 터미널 명령어를 통해 이 워크플로우를 사용하세요.

## 1. 작업 시작 시 (컨텍스트 로드)
이 스크립트를 실행하면 현재 브랜치의 최근 커밋 및 변경 내역을 바탕으로 작업 컨텍스트를 복구하는 안내를 제공합니다.

명령어:
```bash
./.agent/scripts/load-recent-changes.sh
```

## 2. 작업 종료 시 (자동 커밋 생성 등)
작업이 끝났거나 일정 구간이 완료되었을 때 실행하면, 모든 변경 사항을 자동으로 스테이징(`git add -A`)하고 커밋 메시지를 생성 후 커밋 및 CHANGELOG를 작성합니다.

명령어:
```bash
./.agent/scripts/commit-session.sh
```

> [!TIP]
> Antigravity에게 다음과 같이 지시하여 워크플로우를 자동화할 수 있습니다:
> *"작업이 끝났으니 `git-automation` 워크플로우에 따라서 세션을 커밋하고 종료해줘."*
